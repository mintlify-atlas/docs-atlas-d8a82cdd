## Building RL Environments with OpenEnv

OpenEnv is a framework that treats reinforcement learning environments as microservices — isolated, versioned Docker containers communicating over WebSocket. This course teaches you to use, build, deploy, and train LLMs with OpenEnv from first principles to production.

**5 modules · ~45–60 min each · Concepts + Jupyter notebooks**

## The core idea

OpenEnv makes RL environments look like any other web service:

```python
from envs.openspiel_env import OpenSpielEnv
from envs.openspiel_env.models import OpenSpielAction

with OpenSpielEnv(base_url="https://openenv-openspiel-catch.hf.space").sync() as env:
    result = env.reset()
    result = env.step(OpenSpielAction(action_id=1, game_name="catch"))
    print(result.observation.legal_actions)
```

Every environment — whether hosted on HF Spaces, running in Docker locally, or built by you — uses the same three methods: `reset()`, `step()`, `state()`.
